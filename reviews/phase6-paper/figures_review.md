### Main-Text Figures (8 recommended, WRR typically allows 8-10)

#### Figure 1. Site Map
**What:** Map of CONUS showing 405 sites. Training (260) = circles, Holdout (78) = triangles, Vault (36) = squares. Color by median SSC (log scale). Insets: (a) SSC distribution histogram, (b) per-site sample count distribution.
**Data source:** `dashboard_data/site_metadata.parquet`
**Why it earns its slot:** Required orientation figure. Shows geographic and SSC diversity.
**Axis labels:** Longitude (x), Latitude (y). Colorbar: Median SSC (mg/L, log scale).
**Status:** Likely already generated by dashboard. Verify partition labels are correct for v11 (78 holdout, not 76).

#### Figure 2. Per-Site R2 Distribution (NEW — not in v1 draft)
**What:** Histogram of per-site R2 for 78 holdout sites. Vertical dashed lines at R2 = 0 (red) and R2 = 0.5 (green). Annotations: "24% of sites R2 < 0" and "36% of sites R2 > 0.5". Optional: overlay KDE. Color bars by dominant geology class.
**Data source:** `dashboard_data/per_site.parquet`
**Why it earns its slot:** THE single most important figure for the paper. Shows the full story: bimodal distribution, where the model works and where it fails. Currently missing from the draft.
**Axis labels:** Per-Site R2 (x), Number of Sites (y).
**Status:** MUST BE CREATED. Dashboard diagnostics page may have a version; adapt for print.

#### Figure 3. Geology Box Plot
**What:** Box plot of per-site R2 by dominant geology class (carbonate, sedimentary, metamorphic, volcanic, unconsolidated, mixed). Order by median R2 descending. Individual points overlaid (jitter). N per class annotated.
**Data source:** `dashboard_data/per_site.parquet` joined with geology from `site_metadata.parquet`
**Why it earns its slot:** Directly tests the central hypothesis. Shows carbonate R2 = 0.81 vs volcanic R2 = 0.20.
**Axis labels:** Geology Class (x), Per-Site R2 (y).
**Status:** Dashboard diagnostics page has version. Adapt for print with larger fonts.

#### Figure 4. Adaptation Curve (3 split modes)
**What:** Line plot of MedSiteR2 vs N calibration samples for random, temporal, and seasonal split modes. Shaded regions for bootstrap 95% CIs on random mode. Horizontal dashed line at zero-shot (0.401). Mark the plateau region at N=10.
**Data source:** `v11_extreme_eval_summary.json` adaptation section
**Why it earns its slot:** The paper's strongest practical contribution. The temporal collapse at N=10 is visually dramatic and scientifically important.
**Axis labels:** Number of Calibration Samples (x), Median Per-Site R2 (y).
**Note:** The temporal line crossing below zero-shot at N=10 is the key visual.
**Status:** Dashboard adaptation page has version. Add CIs and styling for print.

#### Figure 5. CatBoost vs OLS at Low N
**What:** Paired bar chart or line plot showing CatBoost vs OLS MedSiteR2 at N = 2, 5, 10, 20, 50 under temporal splitting. Or: slope graph connecting CatBoost and OLS at each N.
**Data source:** OLS benchmark data from `dashboard_data/ols_ols_benchmark_summary.json`
**Why it earns its slot:** Demonstrates the Bayesian shrinkage advantage. The N=2 temporal gap (CatBoost 0.36 vs OLS -0.56) is the paper's strongest operational result.
**Axis labels:** N Calibration Samples (x), Median Per-Site R2 (y). Legend: CatBoost, Per-Site OLS.
**Status:** Dashboard has version. Restyle for print.

#### Figure 6. Brandywine Daily Load Time Series
**What:** Two panels. Top: Full 8-year time series of daily sediment load (tons/day) for USGS 80155 (black), v11 CatBoost (blue), and OLS (red/gray). Log-y axis. Bottom: Zoomed 3-month window around a major storm showing hysteresis capture.
**Data source:** `dashboard_data/load_daily_loads_USGS-01480617.parquet`
**Why it earns its slot:** The Brandywine result is the paper's headline. The time series shows both the total match and the daily scatter, addressing the 2.6% vs 59% discrepancy visually.
**Axis labels:** Date (x), Daily Sediment Load (tons/day, log scale) (y).
**Status:** Dashboard loads page has version. Add event markers and legend.

#### Figure 7. Storm Event Load Scatter (3 sites)
**What:** Three-panel scatter plot (one per site). Each point = one storm event. X-axis = USGS 80155 event load, Y-axis = predicted event load. CatBoost (blue dots) and OLS (red dots) on same axes. 1:1 line (dashed black). 2:1 and 1:2 lines (dotted gray).
**Data source:** `dashboard_data/load_event_loads_*.parquet`
**Why it earns its slot:** Directly shows the event-level advantage. The tighter clustering of CatBoost around 1:1 vs OLS's scatter is immediately visible. Includes all 3 sites including Ferron.
**Axis labels:** USGS 80155 Event Load (tons, log scale) (x), Predicted Event Load (tons, log scale) (y).
**Status:** Dashboard loads page has version. Combine into 3-panel figure.

#### Figure 8. SHAP Beeswarm (Top 15 Features)
**What:** SHAP beeswarm for top 15 features. Each dot = one sample. X = SHAP value, color = feature value (blue low, red high). Show: log_turbidity, turbidity, collection_method, Q_ratio, sgmc features.
**Data source:** `dashboard_data/shap_values.parquet`, `shap_importance.parquet`
**Why it earns its slot:** Shows the three information channels (turbidity, geology, collection method) and demonstrates the model is physically interpretable, not a black box.
**Axis labels:** SHAP Value (x), Feature Name (y). Colorbar: Feature Value (low to high).
**Status:** Dashboard performance page has this. Reduce to 15 features for print readability.
